# Student Management System (SQL + Python)

## Overview
This project is a Student Management System that demonstrates the use of SQL relational databases integrated with a Python application. The system allows users to manage students, courses, and enrollments.

##  Purpose
The purpose of this project is to demonstrate:
- Database creation and management
- CRUD operations (Create, Read, Update, Delete)
- Use of SQL queries including JOIN and aggregate functions
- Interaction between software (Python) and a relational database

## 🛠 Technologies Used
- Python
- SQLite (Relational Database)
- SQL

##  Project Structure
project-folder/
│
├── app.py # Main application
├── database_setup.sql # SQL script to create tables
├── student_management.db # Database file (auto-created)
└── README.md # Project documentation

## ⚙️ How to Run the Project
1. Install Python (if not installed)
2. Open terminal in the project folder
3. Run the program:
4. The database will be created automatically

## Features
- Add students
- Add courses
- Enroll students into courses
- View all students
- View enrollments (using JOIN queries)
- Update student information
- Delete student records
- Perform aggregate operations (COUNT, AVG)

## Example Output
- Display list of students
- Display enrolled courses per student
- Show total number of students
- Show average age of students
